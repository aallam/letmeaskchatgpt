# Let Me Ask ChatGPT

For when someone asks you something they could just ask ChatGPT.

Generate a shareable link with a question. When opened, the link animates typing the question into a ChatGPT-like UI, then redirects to [chatgpt.com](https://chatgpt.com) with the question pre-filled.

Live at [letmeaskchatgpt.aallam.com](https://letmeaskchatgpt.aallam.com)

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```
